Hyper Links

Tag âncora <a>

São os famosos hyperlinks, e a tag usada para se criar um é a <a href=""> </a>, responsável por ligar várias páginas.
Em sua anatomia recebe atributos globais (title, id ,class,...) e outros mais importantes:

<href="">
A referência, podendo ser url completa, fragmento, email, telefone e mais diversos outros.

<download="">
se você colocar o link de uma imagem no href="" com o atributo de download, a imagem será baixada ao invés de irmos até a página e se colocarmos um valor no atributo de download,
ele irá dizer como você vai salvar o arquivo.

<target="">
irá dizer qual o alvo do nosso disparo, ou seja, se não tivermos o target, quando clicarmos no link, ele vai pegar o mesmo alvo,
a mesma página e apenas vai trocar o conteúdo, mas se usarmos o atributo, por exemplo target= "_blank", ele vai abrir uma página com o conteúdo.



Fragmento como referência href
Aje como um botão de âncora que leva até o elemento com o id de mesmo nome que o fragmento (sem o #)



Conteúdos dentro do elemento <a>
Podem ser utilizadas outras tags dentro da tag anchor os quais todos irão agir como redirects para a referência em questão

Tabela básica

Utiliza as tags <table>, <caption>, <thead>, <tbody>, <tfoot>, <tr>, <th>, <td>

<table> = tabela
<caption> = descrição
<thead> = table head
<tbody> = table body
<tfoot> = table foot
<tr> = table row (linha da tabela)
<th> = table header ( cabeçalho - destaca em negrito)
<td> = table description (descrição da tabela)

Todos os conteúdos são organizados por linha na ordem em que aparecem os códigos

Configurações do Cabeçalho <head>

<meta>

Define metadados, como codificação de caracteres especiais e portabilidade para dispositivos mobiles.
Meta normalmente virá com o atributo name para especificar a meta, content para conteúdo, mas há também como o atributo charset, para caracteres especiais.
Já são colocadas automaticamente com o emmet "!"
Geralmente composta pelas variáveis name e content.

<meta charset="UTF-8"> Character set
Possibilita a portabilidade de carcteres especiais na página (ex. Japoneses)
<meta name="viewport" content="width=device-width, initial-scale=1,0">
Possibilita a adaptabilidade da página para a escala da tela de dispositivos móveis. Viewport - Parte visível da página (body)
Define o nome como viewport (name="viewport") (Parte visível)

Favicon


Dispositivos móveis, principalmente Apple possuiem diversas rel diferentes (pesquisar ou utilizar frameworks)


<meta> para SEO (Search Engine Optimization)

Atributos para tag meta que contribuem para o SEO do site

<meta charset=...>

<meta name="author"

Define o autor da página, para possuirmos propriedade sobre a página.

<meta name="description"

Este meta é usado para descrição de sites, caso você não tenha esse meta, o navegador irá procurar qualquer texto seu, mas muito melhor escolher o que você quer que apareça.

<meta name="robots"

Esse meta diz para o robô do google o que queremos que ele faça, ele é responsável por colocar os resultados da busca , por exemplo.
É possível dizer ao robô seguir links na página, através do follow, ou o contrário com o nofollow , ou "indexar" a página, através do index, ou no index.

Meta Social

Existem metadados personalizados por empresas de redes sociais, como Facebook, que criou o Open Graph, que é um tipo de metadado
se quisermos colocar um tipo de conteúdo especial, caso queiramos compartilhar o link da nossa página no Facebook.
