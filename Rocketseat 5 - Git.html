Introdução

O que é Git?

O Git representa um controle de versão do seu código.

Na prática, se cria a partir dele uma linha do tempo contendo backups de versões antigas do código, possibilitando seu manuseio sem preocupações.

Conceitos básicos

  • Repository
    • Local onde ficará o histórico do seu projeto.
  • Branch
    • Linha do tempo
  • Commit
    • Pontos na história
  • Stage
    • Preparação do que será enviado para o ponto na história

Comandos Básicos

                git init        // inicia o git (repositório) no seu projeto

                git add .       // adiciona todos os arquivos modificados ao stage

                git commit -m   // cria e descreve um ponto na história

                git push        // envia alterações para o repositório remoto

GitHub

O que é?

Plataforma online que, a partir do Git, salva e armazena seus projetos de forma remota (nuvem).

Permite a colaboração entre diversas pessoas em diversos projetos, sendo perfeito para trabalhar com times.

Também acaba sendo um portifólio na internet para seus projetos.
