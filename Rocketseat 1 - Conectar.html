Computador, software e hardware


Hardware

Hardware é um termo técnico que se refere à parte física decomputadores e outros sistemas micro eletrônicos.

Software

Software é um conjunto de instruções que permite controlar um aparelho eletrônico,
sendo também considerado como a parte lógica do computador.

Firmware

Conjunto de hardware e software que armazena todas as informações
para que um dispositivo funcione corretamente.

Drivers

Programas de computador (software) que têm a função de conectar o sistema operacional com os
dispositivos eletrônicos, atuando como pontes em termos lógicos, de "como utilizar."

Sistema Operacional

É o software mais importante do computador, interpreta, gerencia e aloca funções
para o hardware e programas no dispositivo.

Programas

Conjunto de instruções que descrevem uma tarefa ou trabalho específico no seu computador,
desenhadas para que o usuário realize alguma ação específica.


Memória RAM

Espaço temporário para informações do sistema operacional e de aplicativos em uso no momento.
Após a conclusão da tarefa os arquivos são movidos para o HD/SSD ou excluídos para entrada de nova data temporária.

HD/SSD

Dispositos de armazenamento de dados. O HD utiliza pratos mecânicos em um cabeçote móvel
para gravar informações a partir da polaridade marcada nesses pratos. Já o SSD armazena dados a partir
de chips de memória, sendo mais rápido e eficiênte a leitura e gravação de informações.

Processador

Considerado como o cérebro do computador. Tranforma informações em uma linguagem que o computador entende (01101110) e vice-versa.
Transforma dados em informações, carrega páginas, faz downloads, abre arquivos e executa programas.


Internet


O que é Internet?

Redes interligadas pelo mundo. Uma rede tem como objetivo interligar computadores para ocorrer o acesso e troca de informações.
Em escala global, somos conectados por cabos sejam eles regionais ou até entre países por cabos submarinos.

IP e MAC Adress

Numeros de identificação dos dispositivos na internet. IP (Internet Protocol) é um número identificador que é dado ao dispositivo
pelo provedor da internet no momento da conexão à rede, sendo mutável dependendo da necessidade. MAC Adress é um número de identificação de série gravado no dispositivo e não muda.

Servidores

Servidor -> Servir com serviços

Computador equipado com mais processadores, portas de comunicação, softwares e sistemas de armazenamento de dados.
Fornece serviços diferentes a uma rede de computadores, chamadas de clientes (Clients). Podem ser hospedagem de sites, provedor de emais, drives, etc.
Também possuem IPs transformados em domínios que determinam sua localização caso seja requisitado alguma informação a partir de seu IP.

DNS

Domain Name System (Sistema de Nome de Domínio). Responsável por traduzir os domínios para números IP, possibilitando a navegação nominal pela internet.

HTTP

Hypertext Transfer Protocol (Protocolo de transferência de Dados Hipertexto). Evento da requisição e envio de dados entre client e servidor.
HTTPS (Hypertext Transfer Protocol Secure). Variação que criptografa o requerimento e o envio dos dados entre os dispositivos.


Sistema Operacional


O que é o Sistema Operacional?

Interface entre o usuário e máquina. Programa que possibilita o controle do usuário ao aparelho, gerenciando dispostivos, aplicativos,
programas e telas.

Características e Objetivo

Fácil entendimento para o usuário, gerenciamento de hardware (periféricos, memória, HD), gerenciamento de softwares (programas).

Grupos de Sistemas Operacionais

Unix

  • Linux
  • OS X
  • Anroid
  • iOS

Windows

  • Windows 7
  • Windows 10
  • Windows Server

Tipos de Face do Usuário

GUI (Graphical User Interface)

Interface gráfica, elementos gráficos.

  • Cursor do Mouse ou touch
  • Área de Trabalho
  • Criar arquivos e pastas de forma interativa

CLI (Comand Line Interface)

Linha de comando (CMD, VSC)

  • Emitir comandos em textos para o computador
  • Criar arquivos e pastas a partir de linhas de códigos

Módulos do Sistema Operacional

Kernel (Cerne)

Componente central do sistema operacional. Primeiro módulo a ser iniciado ao ligar o PC e permanece executando quando o S.O estiver ligado.
Gerencia a memória, processos, armazenamento e dispostivos.

Módulo de Gerenciador de Processos

Processo é um programa em execução. Analisa e divide entre os threads para maior performance

Módulo de Gerenciador de arquivos (File System)

Organização e armazenamento de arquivos (vídeos, imagens, documentos, diretórios)

Tipos de sistemas de gerenciamento de arquivos

  • FAT, NTFS
  • ext3, ext4
  • HFS+, APFS

Ferramentas de Gerenciamento

Tarefas (Task manager, activity monitor)
Pacotes (chocolatey, apt, snap)


Linguagem de Programação


O que é codar

Escrever para o computador, transformando linguagem humana para interpretável à maquina.
É importante seguirmos regras de comunicação, assim como em qualquer lingua falada, sendo nesse caso a partir do código.

Linguagem de Programação

"Idiomas" existentes para dar instruções ao computador.
Conceito de Abstração: tranformação de binário para linguagens, interpretáveis a nós e ao programa após transferência para binário.

Low-level e High-level

Low-level

Linguagens com baixo nível de interferência de tradução de máquina para usuário.

  • Mais difícil de escrever e de ler
  • Mais rápida de se processar pelo PC
  • Origem da programação (Ex. Assembly)

High-level

Linguagens com alto nível de interferência de tradução de máquina para usuário.

  • Mais próxima da comunicação humana (if, else, function, object)
  • Leva mais tempo para a máquina traduzir e entender
  • JavaScript, Python, Rust.

Sintaxe

Conjunto de regras de escrita. Varia de acordo com cada linguagem de programação, tendo seus próprios símbolos e tags.


Tipos de aplicações WEB


O que é uma webpage?

Página na WEB acessada a partir de uma requisição de um client a um servidor com o conteúdo a partir
de seu endereço universal URL (Uniform Recourse Locator). A resposta dada pelo servidor é uma cópia da página,
a qual é gerada pelo navegador a partir do código da página enviada pelo servidor ao client, contendo este:

  • HTML (Hyper Text Markup Language)
  • CSS (Cascading Style Sheet)
  • JavaScript

Website - estático e dinâmico

Website se caracteriza pelo agrupamento estruturado de várias webpages.

Estático

  • A página sempre é a mesma
  • Só se altera com mudança no código
  • Não interage com banco de dados

Dinâmico

  • Conteúdo dap página é mutável
  • Interage com banco de dados, alterando uma página estática com informações que variam de acordo com a solicitação
  • Modificações já configuradas no código
  • Página customizada criada a partir da alteração de uma página padrão
  • Perfis em redes sociais, "meu canal"

Web Application (Aplicação WEB)

Website muito mais complexo, sendo totalmente adaptativa, variável e única para cada usuário.
Ex. Facebook, Gmail, Youtube.
Acabam apresentanto muitos desafios na programação, como Adaptação em diversos dispositivos (Responsividade),
performance e acessibilidade.

Front-end vs Back-end

Front-end

  • Client Side
  • Relação client com browser/app
  • HTML (Mercação), CSS (Personalização), JS (interatividade)
  • Trata da qualidade, faciidade, experiência de interação do client com o conteúdo.

Back-end

  • Server-side
  • Relação browser/app com o servidor/fonte de dados
  • PHP, Java, Python, Java, JavaScript
  • Banco de dados, segurança, processamento de dados, otimização.
  • Trata dos passos por trás das partes visiveis.

WEB Tradicional e SPA (Single Page Application)

WEB Tradicional

Requer atualização (novo request) para ter seu conteúdo atualizado, sendo Front-end (Client-side) e Back-end (Server-side) na mesma aplicação.

SPA

  • Sensação de estar em uma única página
  • Apenas recarrega o conteúdo a ser atualizado
  • Front-end e Back-end são aplicações diferentes
  • Servidor responde primeiro ao Front-end (Client-side) e depois o Back-end é contatado a partir de uma API (Application programming interface)
    a qual age como intermediador de solicitação e requisita ao servidor apenas o conteúdo desejado na alteração (geralmente em fortato json - arquivo de texto)
