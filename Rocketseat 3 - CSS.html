CSS (Cascading Style Sheets)

Folha de Estilo em cascata
Apresentação visual para o cliente

Composto de 'Declarations', pedaços de código que irão ditar as propriedades e valores a serem aplicadas em um elemento HTML.

Como escrevemos?


body { = seletor (Seleciona a tag a ser alterada no HTML)
background: = propriedade (característica a ser mudada)
black; } = valor (mudança da característica)
/* */ = comentários (linhas de códigos ignoradas)


Cascata e Especifidade

Cascata - Quando há 2 (ou mais) declarações, a última será mais relevante




Especifidade - É possível determinar especificidade para tags html a partir de atributos (class, id), tornando-as únicas e as separando mesmo contendo a mesma tag.



Mesmo com todas sendo tags <p>, a ideia de relevância do último não se aplica por causa da especificidade das tags pelos atributos

ATENÇÃO = CSS NÃO RECONHECE ATRIBUTOS CARACTERES INICIAIS DE NÚMEROS OU SÍMBOLOS ESPECIAIS


Box Model - Conceito de Caixas

Tudo são caixas

Todos os elementos do html que serão editados serão considerados como uma caixa.
Essas caixas possuem diversas características, como:

  • Conteúdo
  • Largura
  • Altura
  • Borda
  • Preenchimento interno
  • Preenchimento externo
